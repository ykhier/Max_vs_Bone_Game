# 🐶 Max vs Bones – 2D Platformer Game

A simple and interactive 2D platformer game developed using **Processing (Java)**.

In this game, the player controls a dog ("Max") and must catch falling bones to earn points while avoiding losing lives.

---

## 🎮 Gameplay
- Move left and right using the arrow keys  
- Catch falling bones to gain points  
- Missing a bone reduces your lives  
- The game ends when all lives are lost  
- Reach 10 points to win  

---

## 🚀 Features
- Real-time movement and collision detection  
- Score tracking system  
- Life system with visual hearts  
- Sound effects for success and failure  
- Win and Game Over states  

---

## 🛠️ Technologies
- Processing (Java)  
- Object-Oriented Programming (OOP)  
- Basic game development principles
